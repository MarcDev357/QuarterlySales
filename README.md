# Quarterly Sales Management Web App

A full-stack business web application built with ASP.NET Core MVC.

## Features
- Employee management system
- Quarterly sales tracking
- User authentication and account management
- Custom validation rules for business logic
- Repository pattern for clean data access
- Entity Framework database integration
- MVC architecture for scalable code

## Tech Stack
- ASP.NET Core MVC / C#
- Entity Framework Core
- SQL Server
- Bootstrap

## How to Run
1. Clone the repo
2. Open in Visual Studio
3. Run database migrations: `dotnet ef database update`
4. Launch with IIS Express
